Mobile Sales Data Visualization
Project Overview
This Power BI project provides a comprehensive analysis of mobile phone sales data. The dashboard transforms raw transaction records into actionable insights, helping stakeholders track revenue, identify top-performing brands, and understand customer purchasing behavior across different regions.

Key Features
Sales Performance Tracking: Real-time tracking of Total Revenue, Units Sold, and Average Order Value.

Brand Analysis: Comparative analysis of market share for major brands (Apple, Samsung, Xiaomi, etc.).

Geographic Insights: Visual breakdown of sales performance by city and region.

Time Intelligence: Month-over-Month (MoM) and Year-over-Year (YoY) growth calculations using DAX.

Price Segmentation: Analysis of sales distribution across Budget, Mid-range, and Flagship categories.

Tools & Technologies Used
Microsoft Power BI Desktop (Visualization & Reporting)

Power Query (Data Cleaning & Transformation)

DAX (Data Analysis Expressions) (Complex calculations for time intelligence and aggregation)

Dataset Details
The analysis is based on a dataset containing:

Transaction Dates: To analyze seasonal trends.

Product Details: Brand, Model, and Specifications.

Financial Metrics: Price per unit, Discount, and Total Sale Amount.

Customer Demographics: Location and Payment method.

How to Use
Download the .pbix file from this repository.

Open the file in Microsoft Power BI Desktop.

Use the interactive slicers on the left panel to filter data by Brand, Year, or Region.
